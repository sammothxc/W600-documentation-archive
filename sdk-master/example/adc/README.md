## W600 adc example

W600 does not have an external adc pin, but we can use internal adc for temperature measurement

Step 0: Compile and Download

``` 
wch@wch-pc /cygdrive/d/Project/sdk/example
$ ./build.sh adc flash COM3

start...

```
